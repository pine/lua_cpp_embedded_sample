Lua embedded sample for C++
---------------------------

Lua の C++ への組み込みサンプル

## 動作確認環境

- Ubuntu 14.04 LTS (x64)
- Lua 5.1

## ビルド方法
以下は、Ubuntu 14.04 の場合

```
$ sudo apt-get install make g++
$ sudo apt-get install liblua5.1-0-dev
$ make
```
